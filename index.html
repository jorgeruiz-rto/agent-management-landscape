How to read the territory. The orange center is the AI gateway zone: every vendor inside it ships an AI gateway, and the zone deliberately overlaps the API incumbents, the AI-natives, the hyperscalers, and the governance platforms, because that is where Gartner's $250M-to-$1B gateway market is being contested. Gravitee sits at its center as the only neutral, enterprise-grade plane spanning API, event, model, MCP, and agent traffic. IBM occupies a triple point (API incumbent, AIGP Leader, gateway provider) and Airia is the governance vendor that crossed into the zone with an MCP gateway. Salesforce + MuleSoft is drawn as one force straddling the context economy and API management. The agent frameworks band on the eastern flank (LangChain, CrewAI, LlamaIndex) overlaps the open source world and reaches toward the hyperscalers, whose own frameworks (AutoGen and Semantic Kernel, Strands, ADK) make Microsoft, AWS, and Google framework players too: frameworks build the agents, and every agent they spawn is traffic someone must govern. The AI and API security front at the south (Cloudflare, Lasso, Cequence, Palo Alto) enters from risk mitigation, as Gartner's gateway report describes, watching traffic rather than running it. The data and context economy on the western flank owns the context agents consume. Between all of them runs the traffic, and that seam is Gravitee's territory.
